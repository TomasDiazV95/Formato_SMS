Generador de Carga IVR CRM

Sube tu Excel base y configura fecha/rango horario; descarga el Excel final con el formato CRM.

Debe contener columnas: RUT NRO_DOCUMENTO/OP TELEFONO
Sube un archivo Excel válido.
Formato: AAAA-MM-DD
Indica la fecha de gestión.
Ej.: 09:00 o 09:00:00
Indica la hora de inicio.
Ej.: 13:00 o 13:00:00
Indica la hora de fin (mayor que inicio).
Si lo dejas vacío, se auto-ajusta para que todos los registros quepan dentro del rango.
Selecciona un USUARIO.
Escribe el texto que quieras registrar o déjalo vacío.
Orden de columnas: RUT | NRO_DOCUMENTO | FECHA_GESTION | TELEFONO | OBSERVACION | USUARIO | CORREO.
Notas
  • Las columnas RUT, NRO_DOCUMENTO (detectado como OP/Operación/Nro Documento) y TELEFONO se toman tal cual del Excel origen.
  • OBSERVACION guarda exactamente lo que escribas (puede quedar en blanco).
  • USUARIO será el seleccionado; CORREO se completa con un espacio (" ").
  • Si defines un intervalo, no se sobrepasará la hora fin. Si el intervalo no alcanza para todos, se te avisará.
  • La hoja se llamará Hoja1 y el archivo cargaIVR_CRM_dd-mm.xlsx.
